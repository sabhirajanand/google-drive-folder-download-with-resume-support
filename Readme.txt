With this python project you can download all files located in your google drive folder in one go. 

This also supports resuming download in case the download was interrupted.

This is implemented using Google Drive API provided by Google Cloud.

Requirements :

1.install python 3.9 or above

2.use "pip install <module_name>" to install below modules:   

i)   googleapiclient 
ii)  google_auth_oauthlib
iii) google.auth
iv)  google.oauth2
